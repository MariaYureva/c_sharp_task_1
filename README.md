# Задание 1. Вариант 7. Класс «Студент»

Оконное приложение на C# (WPF) с использованием архитектуры MVVM.  
Программа демонстрирует работу класса **Student** (Студент) с оценками по семестрам.

---

## 📌 Описание проекта

В проекте реализован класс `Student`, который хранит информацию о студенте:

- Фамилия
- Имя
- Дата рождения
- Курс
- Группа
- Список предметов по семестрам
- Оценки (через индексатор с двумя индексами)

Приложение позволяет:

- Просматривать данные студента
- Вводить и изменять оценки
- Вычислять средний балл
- Получать среднюю оценку по выбранному предмету
- Просматривать задолженности

---

## 🧱 Архитектура

Используется паттерн **MVVM (Model-View-ViewModel)**:

- **Models**
  - `Student` — бизнес-логика

- **ViewModels**
  - `MainViewModel`
  - `GradeRowViewModel`
  - `ObservableObject`

- **Views**
  - `MainWindow.xaml`
